\section{Πρόβλημα 4}

Στο selfish routing μπορεί να υπάρχουν πολλοί τρόποι να βάλεις βέλτιστα διόδια,
δηλαδή διόδια στις ακμές του δικτύου που κάνουν την βέλτιστη ροή ροή ισορροπίας
στο δίκτυο με τα διόδια.

\subsection{Ερώτηση (i)}

Θα δείξουμε ότι σε $s$-$t$ δίκτυα μπορούμε πάντα να βρίσκουμε βέλτιστα διόδια
όπου ένα $s$-$t$ μονοπάτι να έχει μηδενικά διόδια καθόλο το μήκος του.

\subsection{Ερώτηση (ii)}

Θα εξετάσουμε αν στα βέλτιστα διόδια που βρήκαμε οι παίκτες πληρώνουν συνολικά
το μικρότερο ποσό σε διόδια συγκρινόμενο με όλα τα πιθανά σετ βέλτιστων διοδίων.

\subsection{Ερώτηση (iii)}

Θα εξετάσουμε αν σε δίκτυα με πολλά ζεύγη $s_i$-$t_i$ ισχύει κάτι αντίστοιχο για
κάποιο ζεύγος $s_i$-$t_i$ και αν μπορούμε να επιλέξουμε εμείς το $i$.
